Friends are either functions or classes that you give access to another classes member variables. In order to establish a function as a friend, you include the prototype in the class definition with the friend keyword.
```C++
class Dog {
public:
	Dog(std::string dogName, int dogAge) {
		this->dogName = dogName;
		this->dogAge = dogAge;
	}
	
	// Friend Declarations
	friend void debugDogInfo(const Dog &dog);
	friend void debugDogInfo();
	friend class SomeClass;
private:
	std::string dogName;
	int dogAge;
};
```
